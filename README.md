# projekt-sieci-neuronowe-i-deep-learning
W ramach projektu zdecydowaliśmy się zbudować model, który będzie kla syfikował zdjęcia w celu rozpoznawania obiektów, nawet dla bardziej złożo nych danych. Przy pomocy sieci neuronowych trenujemy model, który może służyć jako filtr w aplikacjach lub może być wykorzystany do maszyn sor tujących, przykładowo owoców i warzyw lub paczek.
